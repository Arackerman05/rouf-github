BIODATA

Halo, perkenalkan nama saya Abdur Rouf, saya seorang mahasiswa semester 2 jurusan Teknik Informatika yang baru saja terjun ke dunia Frontend developer. Dalam waktu lima bulan, saya telah memahami dan aktif menggunakan HTML, CSS, dan JavaScript. Selain mengejar studi, saya juga terlibat dalam proyek pengembangan web di kampus, menghadirkan ide-ide kreatif dan dinamis menggunakan keterampilan yang telah saya pelajari. Perjalanan ini membuat saya semakin antusias, dan saya memiliki tujuan untuk terus mengembangkan diri sebagai seorang Frontend Developer yang handal. Saya aktif mengikuti workshop dan seminar, serta siap berkolaborasi dalam proyek-proyek menarik. Jika ada peluang atau ide-ide baru, jangan ragu untuk menghubungi saya melalui email. Mari bersama-sama menjelajahi dunia teknologi dan meraih impian bersama!
